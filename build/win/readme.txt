[English]
I’m really sorry, but I have to temporarily stop Linux support. 
It’s not because I hate Linux — but I got too many errors and issues 
when trying to build on Linux systems.

If you are experienced, you can add Linux support yourself.  
I will be really happy to see your contribution! :]

I have just one request — for advanced developers:
    ❌ Please don’t use CMake — it makes it too hard for beginners to build their games.  
    ✅ Use simple .bat (Windows) or .sh (Linux) scripts instead.  
    As a last option, Makefile is fine too.

Good luck developing games!  
By LethalStudio and ProtoEngine!
